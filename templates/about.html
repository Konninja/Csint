📖 Intelligence Framework

This platform consolidates three distinct intelligence collection disciplines into a single, unified dashboard. Designed for private investors and commercial due diligence, it enables rapid reconnaissance of digital assets, entities, and infrastructure.

🌐 OSINT OPEN SOURCE

Open Source Intelligence — Collection from publicly available sources with no authentication required.

  • Geolocation — IP address mapping to continent, country, region, city, coordinates, ISP, and ASN via ip-api.com
  • DNS Records — A, AAAA, MX, NS, TXT, CNAME resolution via Google DNS over HTTPS (RFC 8484)
  • WHOIS / RDAP — Domain registration data including registrant, dates, name servers via who-dat.as93.net
  • Subdomain Enumeration — Certificate transparency log querying via crt.sh

📊 CSINT COMMERCIAL SIGNALS

Commercial Signals Intelligence — Collection from semi-public or rate-limited sources that may require registration or API keys but are freely available.

  • Email Reputation — Known spam, phishing, or malicious activity reports via emailrep.io
  • Data Breach Exposure — Credential leaks and past compromise checks via XposedOrNot
  • Technology Profiling — HTTP header analysis, SSL certificate examination, and framework detection
  • Reverse IP Lookup — Other domains hosted on the same IP address via HackerTarget

🔒 CLOSINT CLOSED INDICATORS

Closed Intelligence — Collection from curated threat intelligence platforms that aggregate confirmed malicious indicators from closed communities and automated systems.

  • AbuseIPDB — IP address abuse reports, confidence scores, and report categories (requires free API key)
  • URLhaus — Malware URL and hostname lookup against the abuse.ch threat feed (requires free API key)

🎯 Use Cases

🏢 Investment Due Diligence

Evaluate the digital footprint and infrastructure posture of target companies before acquisition or investment. Identify hosting providers, technology stacks, and potential security liabilities.

🕵️ Competitive Intelligence

Map competitor digital infrastructure, identify subdomains, discover connected services, and profile technology choices to inform strategic positioning.

🔐 Risk Assessment

Check email domains for breach exposure, evaluate IP address reputation, and identify associations with malicious infrastructure through threat intelligence feeds.

📋 Background Verification

Verify domain ownership, registration details, hosting locations, and infrastructure history to validate claims made by counterparties or partners.

⚙️ Technical Architecture

Backend Stack

  • Framework: Python 3.11 + Flask (WSGI via Gunicorn)
  • API Layer: RESTful JSON endpoints, smart target auto-detection
  • External Integrations: 10+ free intelligence APIs with fallback handling
  • Deployment: Render free tier (web service)

Frontend Stack

  • UI Framework: Bootstrap 5.3 with dark theme
  • Typography: Inter (UI) + JetBrains Mono (data)
  • Interactivity: Vanilla JavaScript with async API calls
  • Data Presentation: Structured cards with copy-paste textareas

📜 Legal & Ethics

This tool is designed for legitimate intelligence gathering — including due diligence, security research, competitive analysis, and threat assessment — against targets you have authorization to investigate or that are publicly available information.

Users are responsible for complying with all applicable laws and regulations in their jurisdiction, including but not limited to data protection regulations (GDPR, CCPA, etc.), computer misuse legislation, and terms of service of queried platforms.
